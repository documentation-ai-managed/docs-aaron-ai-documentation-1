## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaborative editing for team members working on the same document
- Introduced custom branding options including logo uploads and color themes
- Implemented advanced search with filters for document type, author, and date ranges
- Added support for embedding interactive code examples directly in documentation

## Improvements

- Enhanced API performance with improved caching and response times
- Updated user interface with better accessibility features and mobile responsiveness
- Streamlined project creation workflow with guided setup process
- Improved webhook reliability with automatic retry mechanisms

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched complete API overhaul with RESTful endpoints for all major operations
- Introduced enterprise SSO integration supporting SAML and OAuth protocols
- Added granular permission system with custom roles and project-level access control
- Implemented automatic documentation versioning with diff tracking

## Breaking Changes

- API v1 endpoints are deprecated; migrate to v2 by October 31, 2024
- Authentication flow updated to require API keys for programmatic access
- Project structure changed to support nested documents and sections

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added webhooks for real-time notifications on documentation changes
- Introduced template system for common documentation patterns
- Implemented document export functionality in multiple formats (PDF, HTML, Markdown)
- Added team collaboration features with comments and mentions

## Bug Fixes

- Fixed issue with document versioning not preserving formatting
- Resolved authentication timeout problems for long editing sessions
- Corrected search indexing delays for newly created content
- Fixed mobile layout issues in the document editor

</Update>
